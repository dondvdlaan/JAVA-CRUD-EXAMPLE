This example applies the MVC architecture:

Model: PKW and User Objects
View: FXML GUI
Controller: Database Dao and DB Manger, CsvFileHandler

Database used is the MariaDB. Communication with the DB is per JDBC.

IDE used: IntelliJ IDEA

Instructions how to use:
- Load from resources/databasefiles the mietwagenverwaltung.sql file in the MariaDB
- Access to database, username 'root', password ''
- Check localhost with PHPMyAdmin
- Start Main
- FXML GuiController.java will show up
- Usename 'User_1' and Password '1234' as per userdata/User.txt to log in
